# ScriptChat

**ScriptChat** is an AI-powered storytelling IDE that maps characters, dialogues, and plot relationships into a dynamic Neo4j graph for visual story development and analysis.

---

## Overview

ScriptChat is an experimental writing environment designed for storytellers, screenplay writers, and narrative designers who want to visualize story structures in real time.

---

## Features

- ✍️ AI-assisted script and dialogue generation
- 🕸 Real-time Neo4j graph visualization
- 📊 Character interaction and relationship mapping
- 🔍 Narrative structure and subplot analysis
- 🧠 Context-aware writing suggestions
- ⚡ Dynamic graph updates
- 🎭 Emotional and conflict tracking

---

## Tech Stack

- Python
- Neo4j
- Flask / FastAPI
- NLP & LLM Integration
- Cypher Query Language

---

## Vision

ScriptChat explores the future of intelligent storytelling by transforming scripts into living relational graphs.
